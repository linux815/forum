Для изменения настроек подключения к серверу отредактируте следующие файлы:
class_database.php и class_user.php
и в Конструкторе изменени настройки на свои
	public function __construct()
	{
		$this->hostname = "localhost"; //Имя хоста
		$this->username = "root"; //Логин
		$this->password = "root"; // Пароль
		$this->dbName   = "forum"; // Имя базы данных
	}
================================================================

Если в bbcode не отображаются изображения, тогда отредактируйте файлы шаблонов. Там где содержится строчка:
define('IRB_BB_PATH', 'http://'. $_SERVER['HTTP_HOST'].'/forum/bbcode');
замените путь на свой.
